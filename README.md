# Qingyang Chen — Academic Homepage

A lightweight, responsive academic homepage for Qingyang Chen (陈青阳), an
undergraduate researcher in computer science at Renmin University of China.

The site is intentionally dependency-free and can be served directly by GitHub
Pages. Open `index.html` locally or visit `https://anapplefish.github.io` after
deployment.

## Privacy

The public site includes only research-relevant information. Phone number,
hometown, ethnicity, political affiliation, detailed grades, and the private CV
are deliberately excluded.
